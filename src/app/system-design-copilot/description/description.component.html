This project is an AI-powered system design copilot designed to assist users in learning and creating software architectures. It provides real-time feedback on design decisions and tradeoffs.The core of the project is an interactive web app built with React which includes a chat interface that enables users to discuss design choices step-by-step. Users can also construct, visualize, and receive feedback on system design diagrams. The backend, implemented with Go and the Azure OpenAI SDK, leverages a chat-stream API to process user queries and stream insightful responses directly to the client, enhancing the learning and design experience.
